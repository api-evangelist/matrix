# MatrixThis is a repo for managing machine readable artifacts for the [Matrix APIs](http://matrix.org/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Matrix APIs](http://matrix.org/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://matrix.org/)- [Developer](http://matrix.org/)- [Twitter](https://twitter.com/matrixdotorg)- [GitHub](https://github.com/matrix-org)The [APIs.json](https://github.com/api-evangelist/matrix/blob/master/apis.json) index provides links to all the relevant links for the Matrix APIs, including the OpenAPI and Postman if they are present.
